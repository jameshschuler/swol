## Swol

### About
Swol is a collection of strength sports tools.

### Features
- One Rep Max Calculator
- Barbell Load Calculator
- Gym Check-In Tracker
- Third-party authentication with Google and GitHub
- PWA

### Technology
- Vue 3 
- TypeScript
- Scss
- Pinia
- Vite
- Vue Router
- Pico.css
- PWA
